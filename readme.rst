###################
SISTEMA DE GESTION DE INCIDENCIAS
###################

Aplicación desarrollada en PHP que permite llevar el registro de cada uno de los despachos que se generan en una empresa.

*******************

1. Acceso al Sistema
2. Incidencias
3. Ejecución

- Registrar Caso
- Ejecutar

4. Seguimientos
5. Avances
6. Informes
7. Configuraciones

- Estados Incidencias
- Usuarios
- Dias Solucion
- Proyectos
- Permisos
- Estados Casos
